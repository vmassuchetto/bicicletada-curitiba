== Description ==

Inserts WP Page Navi, WP Print and WP Post Ratings Plugins in your theme from the Dashboard. No code necessary!
As you know, some plugins need to be installed manually, by editing the theme. Automatic plugins is the easy way to use those plugins without have programming knowledges.
This plugin works only with:
1. WP PAGE NAVI
2. WP PRINT
3. WP POST RATINGS

N.B. You must install those plugins before use them with Automatic Plugins.


== Installation ==

1. Upload `automatic-plugins` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Read the Automatic Plugins page in Settings menu in wp-admin

== Frequently Asked Questions ==

How does this plugin work?

Very easy.
This plugin uses the Wordpress` hooks to modify your theme. It is not dangerous for your theme. This plugin does not affect your post content or layout. It only inserts WP Print Link, Post Ratings and WP Page Navi before or after your posts.

Does this plugin works on all WP Themes?

Yes, we tested it on Twentyeleven Theme and on other free themes. 
Notice that in some cases the desired layout can be appear missaligned.
